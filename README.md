# Drunken Snake Game

A browser-based Snake Game built using HTML, CSS, and JavaScript. This project enhances the classic Snake game with additional features such as a start screen, pause functionality, and game-over interface.

---

## Project Overview

The Drunken Snake Game is an interactive web-based game where the player controls a snake to collect food while avoiding collisions with itself. The game includes smooth controls, score tracking, and multiple UI states such as start, pause, and game over.

---

## Features

* Interactive snake movement using arrow keys
* Start screen with instructions
* Pause and resume functionality
* Real-time score tracking
* Game over screen with final score display
* Continuous gameplay using edge wrapping
* Responsive and simple UI design

---

## Tech Stack

* HTML5 (Canvas for rendering)
* CSS3 (Styling and layout)
* JavaScript (Game logic and interactivity)

## Game Controls

* Arrow Up: Move Up
* Arrow Down: Move Down
* Arrow Left: Move Left
* Arrow Right: Move Right
* Pause Button: Pause/Resume the game
